# Установка FunPayAPI

> **Источник:** https://funpayapi.readthedocs.io/ru/latest/install.html

---

## Способы установки

### ✅ 1. PIP *(рекомендуется)*

```bash
pip install FunPayAPI
```

### 2. pipenv

```bash
pipenv install FunPayAPI
```

### 3. Клонирование репозитория

```bash
git clone https://github.com/woopertail/FunPayAPI.git
cd FunPayAPI
python setup.py install
```

### 4. PIP напрямую из GitHub

```bash
pip install git+https://github.com/woopertail/FunPayAPI.git
```

---

## Обновление

```bash
pip install FunPayAPI --upgrade
```

> ⚠️ Библиотека активно разрабатывается — рекомендуется регулярно обновляться.

---

## Требования

- **Python** 3.10 и выше
- Зависимости устанавливаются автоматически
