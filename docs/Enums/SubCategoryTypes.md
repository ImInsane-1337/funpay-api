# SubCategoryTypes

**Модуль:** `FunPayAPI.common.enums.SubCategoryTypes`

Перечисление типов подкатегорий лотов.

## Значения

| Константа | Значение | Описание |
|-----------|----------|----------|
| `COMMON` | `0` | Обычная подкатегория (аккаунты, услуги, предметы) |
| `CURRENCY` | `1` | Подкатегория игровой валюты (например, адена, золото, серебро) |

> **Важно:** Подкатегории типа `CURRENCY` не поддерживают ручное поднятие лотов через функцию `raise_lots()`.
