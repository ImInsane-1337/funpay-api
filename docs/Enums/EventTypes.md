# EventTypes

**Модуль:** `FunPayAPI.common.enums.EventTypes`

Перечисление всех типов событий, которые могут быть возвращены генератором `Runner.listen()`.

## Значения

| Константа | Значение | Описание |
|-----------|----------|----------|
| `INITIAL_CHAT` | `0` | Обнаружен существующий чат при инициализации |
| `CHATS_LIST_CHANGED` | `1` | Список чатов изменился |
| `LAST_CHAT_MESSAGE_CHANGED` | `2` | Последнее сообщение в чате изменилось |
| `NEW_MESSAGE` | `3` | Обнаружено новое сообщение в истории чата |
| `INITIAL_ORDER` | `4` | Обнаружен существующий заказ при инициализации |
| `ORDERS_LIST_CHANGED` | `5` | Список заказов изменился |
| `NEW_ORDER` | `6` | Обнаружен новый оплаченный заказ |
| `ORDER_STATUS_CHANGED` | `7` | Изменился статус существующего заказа |
