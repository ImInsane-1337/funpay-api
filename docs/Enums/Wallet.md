# Wallet

**Модуль:** `FunPayAPI.common.enums.Wallet`

Перечисление платёжных систем (кошельков) для вывода средств с баланса аккаунта.

## Значения

| Константа | Значение | Описание |
|-----------|----------|----------|
| `QIWI` | `0` | QIWI Кошелёк |
| `BINANCE` | `1` | Binance Pay |
| `TRC` | `2` | USDT (TRC20) |
| `CARD_RUB` | `3` | Банковская карта (Рубли) |
| `CARD_USD` | `4` | Банковская карта (Доллары) |
| `CARD_EUR` | `5` | Банковская карта (Евро) |
| `WEBMONEY` | `6` | WebMoney (WMZ) |
| `YOUMONEY` | `7` | ЮMoney (бывший Яндекс.Деньги) |
