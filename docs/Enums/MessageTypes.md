# MessageTypes

**Модуль:** `FunPayAPI.common.enums.MessageTypes`

Перечисление всех возможных типов сообщений в чате (как системных, так и обычных сообщений).

## Значения

| Константа | Значение | Описание |
|-----------|----------|----------|
| `NON_SYSTEM` | `0` | Обычное текстовое сообщение |
| `ORDER_PURCHASED` | `1` | Системное: Заказ оплачен покупателем |
| `ORDER_CONFIRMED` | `2` | Системное: Выполнение заказа подтверждено |
| `NEW_FEEDBACK` | `3` | Системное: Покупатель оставил отзыв |
| `FEEDBACK_CHANGED` | `4` | Системное: Покупатель изменил отзыв |
| `FEEDBACK_DELETED` | `5` | Системное: Покупатель удалил отзыв |
| `NEW_FEEDBACK_ANSWER` | `6` | Системное: Продавец ответил на отзыв |
| `FEEDBACK_ANSWER_CHANGED` | `7` | Системное: Ответ на отзыв изменён |
| `FEEDBACK_ANSWER_DELETED` | `8` | Системное: Ответ на отзыв удалён |
| `ORDER_REOPENED` | `9` | Системное: Заказ открыт повторно (спор) |
| `REFUND` | `10` | Системное: Произведён полный возврат средств |
| `PARTIAL_REFUND` | `11` | Системное: Произведён частичный возврат средств |
| `ORDER_CONFIRMED_BY_ADMIN` | `12` | Системное: Заказ подтверждён администратором |
| `DISCORD` | `13` | Системное: Ссылка на Discord-сервер (внутреннее системное уведомление) |
