# Account.refund()

Возвращает деньги покупателю по заказу.

## Сигнатура

```python
Account.refund(order_id: str) -> None
```

## Параметры

| Параметр | Тип | Описание |
|----------|-----|----------|
| `order_id` | `str` | ID заказа для возврата |

> ⚠️ Действие необратимо. Используйте с осторожностью.
