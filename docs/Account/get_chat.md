# Account.get_chat()

Получает полную информацию о чате по его ID.

## Сигнатура

```python
Account.get_chat(chat_id: int) -> Chat
```

## Параметры

| Параметр | Тип | Описание |
|----------|-----|----------|
| `chat_id` | `int` | ID чата |

## Возвращает

`Chat` — объект чата. Подробнее: [Types/Chat.md](../Types/Chat.md)
