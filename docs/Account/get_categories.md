# Account.get_categories()

Получает список всех категорий FunPay.

## Сигнатура

```python
Account.get_categories(update_cache: bool = True) -> list[Category]
```

## Параметры

| Параметр | Тип | По умолч. | Описание |
|----------|-----|-----------|----------|
| `update_cache` | `bool` | `True` | Обновить кэш категорий |

## Возвращает

`list[Category]` — список категорий. Подробнее: [Types/Category.md](../Types/Category.md)
