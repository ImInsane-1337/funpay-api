# SubCategory

Класс представляет подкатегорию товаров на FunPay (например, конкретная игра с лотами или валютой).

## Атрибуты

| Атрибут | Тип | Описание |
|---------|-----|----------|
| `id` | `int` | ID подкатегории |
| `name` | `str` | Название подкатегории |
| `type` | `SubCategoryTypes` | Тип подкатегории (например, обычные товары или игровая валюта) |
| `category` | `Category` | Ссылка на объект родительской категории |
