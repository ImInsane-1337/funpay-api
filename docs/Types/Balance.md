# Balance

Класс представляет баланс аккаунта во всех основных валютах.

## Атрибуты

| Атрибут | Тип | Описание |
|---------|-----|----------|
| `total_rub` | `float` | Общий баланс в рублях (включая задержанные средства) |
| `available_rub` | `float` | Доступный для вывода баланс в рублях |
| `total_usd` | `float` | Общий баланс в долларах США |
| `available_usd` | `float` | Доступный для вывода баланс в долларах США |
| `total_eur` | `float` | Общий баланс в евро |
| `available_eur` | `float` | Доступный для вывода баланс в евро |
