# Message

Класс представляет сообщение в чате.

## Атрибуты

| Атрибут | Тип | Описание |
|---------|-----|----------|
| `id` | `int` | Уникальный ID сообщения |
| `text` | `str \| None` | Текст сообщения. `None`, если сообщение содержит только изображение |
| `chat_id` | `int \| str` | ID чата, к которому относится сообщение |
| `chat_name` | `str \| None` | Никнейм собеседника |
| `author` | `str \| None` | Никнейм автора сообщения |
| `author_id` | `int` | ID автора сообщения |
| `type` | `MessageTypes` | Тип сообщения (системное / обычное) |
| `image_link` | `str \| None` | Ссылка на изображение (если есть) |
| `html` | `str` | HTML-код элемента сообщения |
