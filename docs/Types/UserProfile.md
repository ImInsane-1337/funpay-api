# UserProfile

Класс представляет публичный профиль пользователя на FunPay.

## Атрибуты

| Атрибут | Тип | Описание |
|---------|-----|----------|
| `id` | `int` | ID пользователя |
| `username` | `str` | Никнейм пользователя |
| `profile_photo` | `str` | Ссылка на аватарку профиля |
| `active_lots` | `int` | Количество активных лотов в продаже |
| `reviews` | `list[Review]` | Список отзывов покупателей |
| `html` | `str` | Полный HTML-код страницы профиля |
