# Chat

Класс представляет полную информацию о конкретном чате.

## Атрибуты

| Атрибут | Тип | Описание |
|---------|-----|----------|
| `id` | `int` | ID чата |
| `name` | `str` | Никнейм собеседника |
| `looking_link` | `str` | Ссылка на просматриваемый собеседником товар / профиль |
| `looking_text` | `str \| None` | Текст описания просматриваемого объекта |
| `html` | `str` | HTML-код всей страницы чата |
