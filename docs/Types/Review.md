# Review

Класс представляет отзыв, оставленный покупателем.

## Атрибуты

| Атрибут | Тип | Описание |
|---------|-----|----------|
| `stars` | `int` | Оценка (кол-во звёзд от 1 до 5) |
| `text` | `str \| None` | Текст отзыва |
| `reply` | `str \| None` | Ответ продавца на данный отзыв |
| `author` | `str` | Никнейм автора отзыва |
| `html` | `str` | HTML-код элемента отзыва на странице |
