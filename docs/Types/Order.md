# Order

Класс представляет подробную информацию о конкретном заказе (со страницы самого заказа).

## Атрибуты

| Атрибут | Тип | Описание |
|---------|-----|----------|
| `id` | `str` | Уникальный ID заказа (например, `"ABCD1234"`) |
| `title` | `str` | Название заказа |
| `price` | `float` | Стоимость заказа |
| `buyer_username` | `str` | Никнейм покупателя |
| `buyer_id` | `int` | ID покупателя |
| `status` | `OrderStatuses` | Статус заказа |
| `date` | `datetime` | Дата создания заказа |
| `html` | `str` | HTML-код страницы заказа |
