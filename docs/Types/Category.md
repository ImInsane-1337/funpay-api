# Category

Класс представляет категорию товаров на FunPay (например, "Игры", "Скины" и т.д.).

## Атрибуты

| Атрибут | Тип | Описание |
|---------|-----|----------|
| `id` | `int` | ID категории |
| `name` | `str` | Название категории |
| `subcategories` | `list[SubCategory] \| None` | Список вложенных подкатегорий |
