# LotFields

Класс представляет набор полей конкретного лота. Используется для изменения параметров лота (активность, цена, описание, сервер и т.д.) перед сохранением на FunPay.

## Атрибуты

| Атрибут | Тип | Описание |
|---------|-----|----------|
| `lot_id` | `int` | ID лота (0, если лот новый) |
| `fields` | `dict` | Словарь сырых полей формы редактирования лота |

## Методы

### `set_fields(fields: dict)`
Заменяет весь словарь полей.

### `get_fields() -> dict`
Возвращает текущие поля лота.

### `set(field_name: str, value: str | int)`
Устанавливает значение конкретного поля.

### `get(field_name: str) -> str | int | None`
Возвращает значение указанного поля.

## Пример использования

```python
# Получаем поля лота
fields = acc.get_lot_fields(123456)
# Меняем цену и статус
fields.set("price", 99)
fields.set("active", 1)
# Сохраняем изменения
acc.save_lot(fields)
```
