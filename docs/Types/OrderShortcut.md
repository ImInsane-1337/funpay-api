# OrderShortcut

Класс представляет краткую информацию о заказе из списка заказов (покупок/продаж).

## Атрибуты

| Атрибут | Тип | Описание |
|---------|-----|----------|
| `id` | `str` | Уникальный буквенно-цифровой ID заказа (например, `"#ABCD1234"`) |
| `title` | `str` | Название/описание заказа |
| `price` | `float` | Стоимость заказа |
| `buyer_username` | `str` | Никнейм покупателя |
| `buyer_id` | `int` | ID покупателя |
| `status` | `OrderStatuses` | Статус заказа |
| `date` | `datetime` | Дата и время создания заказа |
| `subcategory_name` | `str` | Название подкатегории, к которой относится заказ |
| `html` | `str` | HTML-код элемента заказа в списке |
