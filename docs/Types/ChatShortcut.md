# ChatShortcut

Класс представляет краткую информацию о чате из общего списка чатов.

## Атрибуты

| Атрибут | Тип | Описание |
|---------|-----|----------|
| `id` | `int` | ID чата |
| `name` | `str` | Никнейм собеседника |
| `last_message_text` | `str \| None` | Текст последнего сообщения (не более 250 символов) |
| `last_message_time` | `str \| None` | Время последнего сообщения (строка) |
| `last_message_author_id` | `int \| None` | ID автора последнего сообщения |
| `unread` | `bool` | Флаг непрочитанного чата |
| `html` | `str` | HTML-код элемента чата из списка |
