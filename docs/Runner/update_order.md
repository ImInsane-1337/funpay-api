# Runner.update_order()

Обновляет сохранённое состояние заказа в кэше Runner'а.

## Сигнатура

```python
Runner.update_order(order: OrderShortcut) -> None
```

## Параметры

| Параметр | Тип | Описание |
|----------|-----|----------|
| `order` | `OrderShortcut` | Обновлённый объект заказа |
